# Commit Convention

✨ feat: 새로운 기능 추가
🐛 fix: 버그 수정
♻️ refactor: 코드 리팩토링 (기능 변경 없음)
🎨 design: UI/스타일 수정 (CSS, 레이아웃 등)
💄 style: 코드 스타일 변경 (세미콜론, 공백 등)
✅ test: 테스트 코드 추가/수정/삭제
🔧 chore: 빌드, 패키지 매니저 등 환경 설정
🎉 init: 프로젝트 초기 생성
🚚 rename: 파일/폴더명 변경 또는 이동
🔥 remove: 파일/코드 삭제
📝 docs: 문서 추가/수정 (README 등)
🚑 hotfix: 긴급 수정
